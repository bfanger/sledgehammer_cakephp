
Installation
--------------

Add the "sledgehammer" folder at the same level as the "cake" folder:

 Project folder
 |- app
 |- cake
 |- plugins
 |- sledgehammer
 |- tmp
 |- vendors


Modify your app/config/bootstrap.php to include:

  require (dirname(__FILE__).'/../../sledgehammer/core/init_framework.php');
